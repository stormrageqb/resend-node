# Resend with Express

This example shows how to use Resend with [Express](https://expressjs.com).

## Instructions

1. Define environment variables in `.env` file.

2. Install dependencies:

  ```sh
npm install
# or
yarn
  ```

3. Run Express locally:

  ```sh
npm run dev
  ```

4. Open URL in the browser:

  ```
http://localhost:3000
  ```
